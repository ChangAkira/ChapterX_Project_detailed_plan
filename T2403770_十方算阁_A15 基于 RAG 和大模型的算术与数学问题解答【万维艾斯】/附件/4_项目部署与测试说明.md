# 本地部署与测试

若要在其它电脑上本地部署与测试项目，则：

1. 确保电脑有足够的空间（大约6~8G）；
2. 确保已经安装了ollama，并运行在默认端口11434；
3. 拉取大模型`ollama pull modelscope.cn/Qwen/Qwen2.5-7B-Instruct-GGUF:Q8_0` , `ollama pull modelscope.cn/Qwen/Qwen2.5-1.5B-Instruct-GGUF:Q8_0` ；
4. 设置系统环境变量 `DEMO_ROOT` 指向本地的的 项目 文件夹；
5. 在START文件夹中，根据 `requirements.txt` 安装所需库。首先激活虚拟环境，防止将项目依赖安装到全局 Python 环境中，从而避免潜在的冲突。接下来 `pip install -r` ；
6. 将整个项目文件夹用python httpserver运行在5500端口 `python -m http.server 5500`；
7. 在项目文件夹/START中启动后端程序 `python server.py`
8. 等到有两行“知识库加载完成”显示后，系统正常运行。在浏览器中打开 `http://127.0.0.1:5500` 。
9. 测试上传图片并回答的功能，需要Umi-OCR运行在1224端口，使用Pix2Text策略。
